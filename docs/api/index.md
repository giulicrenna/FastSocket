# API Reference

This section documents the public API of FastSocket using `mkdocstrings`.

## Module map

| Module | Description |
|---|---|
| [`FastSocket` (legacy)](legacy.md) | Backward-compatible legacy API |
| [`FastSocket.core`](core.md) | Configuration and connection handling |
| [`FastSocket.server`](server.md) | TCP and UDP server classes |
| [`FastSocket.client`](client.md) | TCP and UDP client classes |
| [`FastSocket.security`](security.md) | RSA and hybrid encryption modules |
| [`FastSocket.utils`](utils.md) | ChunkManager, FileTransfer, logger, exceptions |
